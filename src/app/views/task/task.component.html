Задание

Создать проект Angular. В проекте будет 2 страницы. главная и страница с формой. Страницу с формой сделать отдельным модулем. Переход на страницу с формой будет по нажатию кнопки с главной страницы. На странице будет кнопка для перехода на главную страницу.

Поля формы:
  • поле для ввода названия метода. (Только латинские символы);
  • выпадающий список для выбора метода отправки запроса (GET, POST);
  • текстовое поле для ввода данных в формате JSON;
  • кнопка "Отправить".

Перед отправкой поля проверяются на правильность. Для Get запроса JSON должен иметь только один уровень. После нажатия на кнопку "Отправить" выбранным методом отправляется запрос на адрес https://scloud.ru/user_panel/test_angular.php?method=<метод> . Время ответа от сервера ограничить 20 сек. При превышении времени выдавать ошибку. При получении ошибки от сервера трижды посылать запрос перед тем как вывести ошибку.

После получения ответа вывести:
  • время ответа выполнения запроса;
  • код ответа;
  • текст ответа(ответ будет в формате JSON) форматированный для удобного чтения.

Если получаем ошибку вывести просто текст "Запрос вернулся с ошибкой" красным цветом.
